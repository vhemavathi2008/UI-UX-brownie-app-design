# UI-UX-brownie-app-design
Designed a simple Brownie Delivery Mobile UI in Figma 🍫📱

This project is based on a brownie ordering app where users can explore delicious brownies, view special offers, and order their favorite desserts easily.

What I practiced:
• UI Design Basics
• Cards & Layouts
• Colors and Spacing
• Basic Figma Prototyping
• Simple Navigation Flow

I also learned how to create simple interactive prototypes in Figma by connecting screens and adding basic transitions.

Currently improving my UI/UX skills step by step through practice and mini projects 🚀
